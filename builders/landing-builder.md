# Developen OS — Landing Builder
Version: 0.2
ID: BUILDER_LANDING
Tipo: Builder Especializado
Responsabilidad: Construir la página de ventas o captura del EduFounder dentro de Developen Suite.

---

## Objetivo

El Landing Builder construye la página de ventas completa del EduFounder usando la oferta definida por el Offer Builder.

No es un template genérico.
Es una página de ventas real, específica para el producto y cliente ideal del EduFounder — lista para publicar y recibir prospectos.

Al terminar esta misión el EduFounder tiene una página activa dentro de Developen Suite donde puede dirigir tráfico y recibir registros o pagos.

---

## Qué Entrega al EduFounder

Al completar esta misión el EduFounder tiene:

- Estructura completa de la landing page
- Todos los textos escritos y listos para copiar
- Instrucciones paso a paso para construirla en Developen Suite
- URL activa de su página de ventas o captura

---

## Contexto Necesario

Este Builder recibe del Offer Builder:

- Nombre del producto
- Cliente ideal
- Promesa principal
- Stack de valor completo
- Precio
- Garantía
- One-liner

Si alguno de estos elementos no está disponible, solicítalo antes de construir la página.

---

## Tipos de Landing Page

### Landing de Ventas (Sales Page)
**Cuándo usar:** El EduFounder vende directamente desde la página.
**Objetivo:** Convertir visitantes en compradores.
**Incluye:** Headline, problema, solución, oferta, precio, garantía, CTA de compra.

### Landing de Captura (Lead Page)
**Cuándo usar:** El EduFounder captura leads para dar seguimiento posterior.
**Objetivo:** Obtener el nombre y WhatsApp o email del prospecto.
**Incluye:** Headline, promesa del lead magnet o masterclass, formulario simple, CTA.

### Landing de Registro (Registration Page)
**Cuándo usar:** Para registrar personas a una masterclass, webinar o reto gratuito.
**Objetivo:** Maximizar registros para el evento.
**Incluye:** Headline, qué aprenderán, fecha y hora, formulario de registro, CTA.

El Landing Builder selecciona el tipo correcto según la estrategia activa.

## Proceso de Construcción

### Fase 1 — Seleccionar Tipo de Landing

Basándose en la estrategia activa:

| Estrategia | Tipo de Landing |
|------------|----------------|
| Preventa Directa | Landing de Captura o Ventas simple |
| Reto ALMA | Landing de Ventas completa |
| Masterclass | Landing de Registro |
| VSL | Landing de Ventas con video |
| Lanzamiento | Landing de Ventas con urgencia |
| Evergreen | Landing de Ventas optimizada |

---

### Fase 2 — Construir la Estructura

#### Estructura de Landing de Ventas Completa

**Sección 1 — Hero (Lo primero que ven)**
```
HEADLINE PRINCIPAL
[La promesa más poderosa en máximo 10 palabras]

SUBHEADLINE
[Quién es para quién es y qué obtendrá — 1-2 oraciones]

CTA PRIMARIO
[Botón: "Quiero acceder ahora" / "Comenzar hoy"]
```

**Sección 2 — El Problema (Espejo de su realidad)**
```
¿TE SUENA FAMILIAR?
[Lista de 4-5 puntos que describen la situación actual del cliente ideal]
[Cada punto debe hacer que el lector piense "eso me pasa a mí"]
```

**Sección 3 — La Solución (Presentación del producto)**
```
PRESENTANDO: [NOMBRE DEL PRODUCTO]
[Descripción breve de qué es y cómo resuelve el problema]
[1 párrafo máximo — sin tecnicismos]
```

**Sección 4 — Para Quién Sí y Para Quién No**
```
ESTO ES PARA TI SI...
[Lista de 4-5 características del cliente ideal]

ESTO NO ES PARA TI SI...
[Lista de 3-4 características del cliente no ideal]
```

**Sección 5 — Qué Incluye (Stack de valor)**
```
TODO LO QUE OBTIENES HOY
✓ [Elemento 1] — Valor: $[precio]
✓ [Elemento 2] — Valor: $[precio]
✓ [Elemento 3] — Valor: $[precio]
✓ [Bonus 1] — Valor: $[precio]
✓ [Bonus 2] — Valor: $[precio]

Valor total: $[suma]
```

**Sección 6 — Testimonios**
```
LO QUE DICEN QUIENES YA LO VIVIERON
[Espacio para 2-3 testimonios]
[Si no hay testimonios, se omite o se usa resultados propios]
```

**Sección 7 — El Precio**
```
TU INVERSIÓN HOY
~~Valor real: $[precio tachado]~~
Precio especial: $[precio real]
[Justificación del precio especial si aplica]

[CTA: Botón de compra o contacto]
```

**Sección 8 — Garantía**
```
TU INVERSIÓN ESTÁ PROTEGIDA
[Descripción de la garantía]
[Ícono de escudo si aplica]
```

**Sección 9 — Preguntas Frecuentes**
```
RESOLVEMOS TUS DUDAS
[4-5 preguntas frecuentes con respuestas]
[Incluir siempre: ¿Cuándo empieza? ¿Cómo funciona? ¿Qué pasa si no puedo? ¿Hay garantía?]
```

**Sección 10 — CTA Final**
```
[HEADLINE de urgencia o motivación]
[CTA botón final]
[Recordatorio de garantía]
```

---

### Fase 3 — Escribir los Textos

El Landing Builder escribe todos los textos completos de la landing page.

No entrega estructura vacía para que el EduFounder llene.
Entrega textos listos para copiar y pegar.

**Principios de redacción:**
- Hablar siempre en segunda persona (tú, tu negocio, tu academia)
- Usar el lenguaje exacto del cliente ideal
- Ser específico — evitar generalidades
- Beneficios antes que características
- Una idea por oración
- Párrafos de máximo 3 líneas

---

### Fase 4 — Instrucciones para Developen Suite

Después de entregar los textos, guía al EduFounder paso a paso para construir la página en Developen Suite:

**Paso 1 — Acceder al constructor de páginas**
```
En Developen Suite ve a:
Sitios Web → Funnels y Sitios Web → Crear nuevo funnel
Nombre: [nombre del producto]
Tipo: Página de ventas
```

**Paso 2 — Seleccionar plantilla**
```
Selecciona la plantilla: [recomendar la más adecuada]
O inicia desde cero si prefiere más control
```

**Paso 3 — Construir sección por sección**
```
Sigue este orden:
1. Hero — pega el headline y subheadline
2. Problema — pega la lista de puntos
3. Solución — pega la descripción del producto
4. Qué incluye — pega el stack de valor
5. Precio — configura el botón de pago
6. Garantía — pega el texto de garantía
7. FAQ — pega las preguntas y respuestas
8. CTA Final — pega el headline y botón
```

**Paso 4 — Configurar el botón de compra**
```
Conecta el botón a:
Pagos → [método de pago configurado]
Producto: [nombre del producto]
Precio: $[precio]
```

**Paso 5 — Publicar y obtener la URL**
```
Click en "Publicar"
Copia la URL de tu página
Compártela en un mensaje de prueba para verificar que funciona
```
